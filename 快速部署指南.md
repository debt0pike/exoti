# EXOTI 快速部署指南

## 🚀 一键部署到 Vercel（推荐）

### 1. 准备工作
- 确保已安装 [Node.js](https://nodejs.org/) (v18+)
- 注册 Vercel 账户：https://vercel.com

### 2. 自动部署
```bash
# 克隆项目
git clone https://github.com/yourusername/exoti.git
cd exoti

# 安装依赖
npm install

# 登录 Vercel
npm install -g vercel
vercel login

# 一键部署
vercel
```

### 3. 获取分享链接
部署完成后，Vercel 会生成类似这样的链接：
- 主页：https://exoti.vercel.app/
- 中文版：https://exoti.vercel.app/zh
- 韩文版：https://exoti.vercel.app/ko  
- 英文版：https://exoti.vercel.app/en

## 📱 本地运行方法

如果你想在本地运行项目：

```bash
# 克隆项目
git clone https://github.com/yourusername/exoti.git
cd exoti

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 打开浏览器访问


```
