## 一、项目概述

**项目名称：** EXOTI（EXO Type Indicator）  
**定位：** 面向EXO粉丝（EXO-L）的人格测评网站，类似MBTI结构，通过27道选择题测出16种EXO-L人格类型  
**参考产品：** BTSTI（BTS Type Indicator）  
**目标用户：** EXO-L，以中文用户为主，兼顾韩文、英文

### 3.2 美学方向

**核心概念：** 黑色宇宙 × 暗金质感 × 星系神话

- **背景：** 深空黑（#0a0a0f）+ 动态星粒子背景（CSS canvas或纯CSS stars）
- **质感：** 轻微噪点纹理叠加，营造胶片感
- **金色：** 用于所有高亮元素（logo、选项边框激活态、进度条、雷达图）

- **星系元素：** 在intro页、result页加入SVG或Canvas星云/星点装饰，呼应EXO宇宙概念

### 5.3 计分逻辑

```typescript
function calculateResult(answers: Record<number, 'A'|'B'|'C'|'D'>) {
  const scores = { A: 0, B: 0, C: 0, D: 0 };
  
  Object.values(answers).forEach(type => {
    scores[type]++;
  });
  
  // 按分数降序排列
  const sorted = Object.entries(scores)
    .sort((a, b) => b[1] - a[1]);
  
  const primary = sorted[0][0];    // 主型（最高分）
  const secondary = sorted[1][0];  // 副型（第二高）
  
  return `${primary}${secondary}`; // 如 "AD"、"BC"
}
```

**16种组合：** AA / AB / AC / AD / BA / BB / BC / BD / CA / CB / CC / CD / DA / DB / DC / DD

### 5.5 分享图片生成

```typescript
// 使用 html2canvas 截图结果卡片
// 生成 1080×1920（竖版）或 1080×1080（方图）可选
// 截图区域单独渲染，不包含导航栏和重测按钮

async function generateShareImage() {
  const element = document.getElementById('share-card');
  const canvas = await html2canvas(element, {
    backgroundColor: '#0a0a0f',
    scale: 2,  // 2x分辨率
    useCORS: true,
  });
  
  const link = document.createElement('a');
  link.download = `EXOTI-${typeCode}.png`;
  link.href = canvas.toDataURL('image/png');
  link.click();
}
```

**分享卡片设计（id="share-card"）：**

- 尺寸：540×960（截图时×2=1080×1920）
- 内容：logo + 人格名 + 中文名 + 一句话 + 雷达图 + 专属曲目 + 底部「EXOTI · exoti.app」水印
- 背景：深空黑+轻微星点

## 八、文件结构

```
exoti/
├── app/
│   ├── [locale]/
│   │   ├── layout.tsx
│   │   ├── page.tsx              ← Intro页
│   │   ├── quiz/
│   │   │   └── page.tsx          ← 答题页
│   │   └── result/
│   │       └── [type]/
│   │           └── page.tsx      ← 结果页
├── components/
│   ├── StarBackground.tsx         ← 动态星空背景
│   ├── QuizCard.tsx              ← 单题组件
│   ├── OptionCard.tsx            ← 选项卡片
│   ├── ProgressBar.tsx           ← 进度条
│   ├── RadarChart.tsx            ← 六维雷达图
│   ├── VinylRecord.tsx           ← 黑胶唱片动画
│   ├── ShareCard.tsx             ← 分享图片卡片（id="share-card"）
│   └── ShareButton.tsx           ← html2canvas分享按钮
├── data/
│   ├── questions.ts              ← 27道题完整数据
│   └── personalities.ts          ← 16种人格完整数据
├── store/
│   └── quizStore.ts              ← Zustand状态管理
├── lib/
│   ├── scoring.ts                ← 计分逻辑
│   └── share.ts                  ← 生成分享图片
├── messages/
│   ├── zh.json
│   ├── ko.json
│   └── en.json
├── styles/
│   └── globals.css               ← CSS变量 + 全局样式
└── public/
    └── fonts/                    ← 字体文件（如需本地化）
```
